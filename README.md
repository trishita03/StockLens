# StockLens
StockLens is a stock market analysis and forecasting application built with Python and Streamlit. It provides recent NVIDIA (NVDA) market information, historical price trends, and a next-session closing-price forecast generated using an LSTM model.
## Features
- Retrieves recent NVIDIA stock data using Yahoo Finance
- Displays the latest closing price and daily percentage movement
- Shows the six-month high and low
- Visualizes historical closing-price performance
- Displays recent OHLC and volume data
- Generates a next-session closing-price forecast
- Shows the projected price change and market direction
- Provides a visual comparison between the latest and forecasted closing prices

## Forecasting Approach

The forecasting component uses a Long Short-Term Memory (LSTM) neural network trained on historical NVIDIA closing-price data.

For each forecast:

1. The latest 60 available trading sessions are retrieved.
2. Closing prices are transformed using the scaler associated with the trained model.
3. The resulting sequence is passed to the LSTM model.
4. The predicted value is inverse-transformed into the original price scale.
5. StockLens compares the forecast with the latest closing price to estimate the projected movement.

## Run Locally

Follow these steps to run StockLens on your local machine.

### 1. Clone the repository

```bash
git clone https://github.com/trishita03/StockLens.git
cd StockLens
```

### 2. Create a virtual environment

```bash
python3.12 -m venv .venv
```

### 3. Activate the virtual environment

On macOS/Linux:

```bash
source .venv/bin/activate
```

On Windows:

```bash
.venv\Scripts\activate
```

### 4. Install the required dependencies
```bash
pip install -r requirements.txt
```
If Streamlit is not installed correctly, you can install it separately using:
```bash
pip install streamlit
```
### 5. Run the application
```bash
python -m streamlit run app.py
```
After running the command, Streamlit will start StockLens and display a local URL in the terminal. Open that URL in your browser to use the application.
